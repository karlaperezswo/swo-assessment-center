SECCIÓN 5 — RIESGOS DE INTEGRACIÓN Y MITIGACIONES
================================================================================

RIESGO 1 — Conflicto de dependencias
  Riesgo: d3, reactflow, jspdf, xlsx pueden tener conflictos con versiones
          ya instaladas en main.
  Mitigación: Verificar package.json de main antes de instalar. Si hay
              versiones distintas, usar la versión de karyn (^7.9.0 para d3,
              ^11.10.4 para reactflow, ^4.2.0 para jspdf).

RIESGO 2 — Conflicto en App.tsx y AssessPhase.tsx
  Riesgo: Estos archivos pueden tener cambios en main que no están en karyn.
  Mitigación: Hacer merge manual línea por línea. Nunca sobrescribir completo.
              Priorizar la lógica de karyn para las secciones de dependencyData.

RIESGO 3 — vis-network CDN en MigrationPlanner
  Riesgo: El grafo de MigrationPlanner carga vis-network desde CDN en runtime.
          Si no hay internet, el grafo no se renderiza.
  Mitigación: vis-network también está en package.json como dependencia local.
              El componente tiene fallback: si window.vis no está disponible,
              inyecta el script desde CDN automáticamente.

RIESGO 4 — Tamaño del bundle
  Riesgo: d3 + reactflow + jspdf + xlsx + vis-network aumentan el bundle.
  Mitigación: Todos usan dynamic import (await import('jspdf')) donde es posible.
              No requiere configuración adicional de Vite.

RIESGO 5 — CORS en backend local
  Riesgo: El backend de main puede tener orígenes CORS distintos.
  Mitigación: backend/src/index.ts de karyn permite:
              localhost:3005, localhost:3006, localhost:5173, 127.0.0.1:3005/3006
              Verificar que el repositorio central use los mismos puertos.

RIESGO 6 — Prop servers en WavePlannerTool
  Riesgo: WavePlannerTool acepta servers como string[] O Server[].
          Si main pasa un tipo diferente, puede causar error en runtime.
  Mitigación: El componente normaliza internamente en el useEffect.
              El guard `if (!serverName) return { apps: [], dbs: [] }` previene crashes.
